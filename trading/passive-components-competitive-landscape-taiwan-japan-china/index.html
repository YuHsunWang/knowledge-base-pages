    被動元件競爭格局：台日中比較¶

    關聯頁面：被動元件產業研究總覽、MLCC、晶片電阻與電感產品矩陣、台灣被動元件公司比較。

    研究原則¶

    被動元件競爭格局不宜只用總市占率判斷，因為 MLCC、晶片電阻、電感、鉭容、薄膜電容、磁性元件的規格層級差異很大。研究重點應拆成：

    • 產品品類。
    • 規格難度。
    • 客戶認證。
    • 製程良率。
    • 成本與規模。
    • 供應穩定度。
    • 是否具備高可靠度紀錄。

    市占率、CAGR、TAM 若來自市調摘要或媒體轉述，僅作線索，不作投資結論。

    日韓廠：高端規格與車規優勢¶

    代表公司：Murata、TDK、Taiyo Yuden、Samsung Electro-Mechanics、Kyocera AVX。

    優勢¶

    • 高容、小型化、低 ESL/ESR MLCC 製程良率。
    • 車規與工控長期認證紀錄。
    • RF、小型化、高頻元件技術積累。
    • 與 Tier-1 / OEM / 高階 ODM 長期合作。
    • 對材料、設備、燒結、薄層化有深厚 know-how。

    研究含義¶

    AI server 高端 MLCC、車規高可靠度 MLCC、RF/高頻小型化元件仍以日韓廠優勢較明顯。台廠若宣稱切入此市場，必須驗證是頂端規格、次高規格，還是補充性供應。

    台廠：平台化、中高階利基與供應彈性¶

    代表公司：國巨、華新科、禾伸堂、大毅、奇力新 / 美桓。

    優勢¶

    • 國巨透過 KEMET、Pulse、奇力新、感測器等併購建立多產品平台。
    • 華新科較直接反映 MLCC / 晶片電阻本業循環。
    • 大毅在晶片電阻與高階電阻升級上具追蹤價值。
    • 奇力新 / 美桓代表台灣電感與磁性元件脈絡。
    • 台廠可能在中高壓、車規次高階、工控、客製化與供應彈性上有機會。

    挑戰¶

    • 頂端 MLCC 小型化與良率仍面對日韓壓制。
    • 標準厚膜電阻、標準 MLCC 面對中國廠價格壓力。
    • AI server 與車用敘事需拆到 AVL / 量產 / 營收占比，不能只看題材。
    • 國巨平台化後研究難度提高，需拆品類與併購資產貢獻。

    中國廠：標準品成本競爭與中高階滲透¶

    代表公司：風華高科、三環、順絡電子等。

    優勢¶

    • 標準品成本競爭力。
    • 中國本地消費電子、通訊與 NEV 客戶供應鏈優勢。
    • 政策與本土替代需求支撐。
    • 部分廠商已嘗試往車規、AI server、功率電感等中高階品項滲透。

    風險與限制¶

    • 頂端 AI server MLCC、高可靠度車規 MLCC、小型化高良率仍需時間追趕。
    • 若只靠價格競爭，容易壓低整體標準品 ASP。
    • 市場傳聞中的供應鏈導入需確認是直接供應、間接供應、送樣或量產。

    美歐廠：精密、高可靠度與特殊品類¶

    代表公司：Vishay、Bourns、KOA、Rohm、Susumu、Isabellenhütte、WIMA 等。

    優勢¶

    • 精密電阻、電流感測電阻、薄膜電阻。
    • 薄膜電容、高可靠度電容。
    • 車用、工控、航太、醫療高可靠度應用。
    • 長期客戶認證與品質紀錄。

    研究含義¶

    台廠若往車用 / 工控 / 高可靠度產品升級，競爭對手不只日韓，也包括美歐精密元件廠。比較時不能只看 MLCC 或厚膜電阻總市占。

    公司比較：研究追問¶

    國巨¶

    • AI 相關營收定義包含哪些品類？MLCC、鉭容、電感、電阻、感測器是否分開揭露？
    • KEMET / Pulse / 奇力新併購後是否形成交叉銷售？
    • 高階品占比提升是否真的降低標準品景氣循環波動？
    • 淨負債、商譽與利息成本是否壓縮估值？

    華新科¶

    • MLCC、電阻、RF/保護元件各占比？
    • AI server / 車用是送樣、AVL 還是量產？
    • 標準品占比是否下降？
    • EPS 是否受到業外或一次性損益影響？

    禾伸堂¶

    • 製造與代理營收/毛利拆分？
    • 代理庫存是否領先反映景氣轉折？
    • 自有品牌 MLCC 是否有利基規格？
    • 毛利率變化是 mix、ASP、代理線或一次性因素？

    大毅¶

    • 高階電阻占比：電流感測、抗硫化、薄膜、高功率。
    • 是否進入車用、AI server power module 或工控客戶？
    • 標準厚膜電阻是否受中國廠報價壓力？
    • 稼動率與毛利率對 ASP 的彈性。

    奇力新 / 美桓¶

    • 功率電感是否進入 AI server VRM 或車用 OBC / DC-DC？
    • 一體成型電感規格與良率如何？
    • 國巨整合後資訊透明度是否下降？
    • 順絡、TDK、Murata 等競爭者在同客戶的定位。

    紅旗風險¶

    • 使用「AI / EV / 車用」標籤，但沒有品類、規格、AVL、量產、營收占比。
    • 市場用未驗證的 GB200 用量或市占率外推公司 EPS。
    • 公司毛利率改善來自短期 ASP 或匯率，卻被解讀成結構性 mix 改善。
    • 標準品中國競爭壓力被 AI / 車用敘事掩蓋。
    • 代理業務與製造業務混合估值，導致毛利率與營收品質誤判。
