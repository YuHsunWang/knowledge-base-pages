    AI Server 被動元件需求驗證框架¶

    關聯頁面：被動元件產業研究總覽、MLCC、晶片電阻與電感產品矩陣、被動元件景氣循環與追蹤框架、台灣被動元件公司比較。

    核心原則¶

    AI server / GPU power architecture 可能是高規格被動元件的結構性需求來源，但不能用單一機架用量或題材字眼直接外推台廠營收。研究上必須拆成：品類、規格、供應鏈資格、量產狀態、營收占比與毛利率貢獻。

    需要避免的錯誤：

    • 不把「GB200 / AI server 被動元件用量」直接等同於「台廠受益」。
    • 不把 sample qualification 誤寫成 production shipment。
    • 不把「通路文章 / 市調摘要」的用量、市占或 CAGR 當作一級事實。
    • 不把「AI 相關營收」與「AI server 直接出貨」混用；公司口徑可能包含資料中心、網通、電源、鉭容、電感或其他高階品。

    需求拆解：品類 × 功能¶

    MLCC¶

    AI server GPU board、CPU board、HBM / memory power、VRM 周邊需要大量去耦與濾波電容。研究重點不是總顆數，而是規格帶：

    • 高容量、小型化。
    • 低 ESL / ESR。
    • 高頻響應。
    • 耐熱與可靠度。
    • 高層數與良率。

    可驗證問題：

    1. 台廠產品是否落在 AI server GPU board 的主流容值 / 耐壓 / 尺寸規格？
    2. 是否已進入 ODM AVL，例如廣達、鴻海、緯創、英業達等 AI server 供應鏈？
    3. 是樣品認證、設計導入，還是量產出貨？
    4. AI server MLCC 毛利率是否高於公司平均？是否有公司法說或 IR 量化？
    5. 訂單是現貨急單、短約、長約，還是 blanket order？
    6. 是否集中於單一 ODM / CSP 客戶，或具備多客戶分散性？

    鉭質 / 聚合物 / 薄膜電容¶

    高功率電源架構需要穩壓、濾波、瞬態電流支援。國巨因 KEMET 併購後具有鉭質、薄膜、鋁電解與高可靠度電容產品線，但仍需拆清楚 AI exposure 來自哪一品類。

    可驗證問題：

    1. 公司揭露的 AI 相關營收中，鉭容、MLCC、電感、電阻各占多少？
    2. 鉭容 AI server 應用是 GPU board、power module 還是資料中心電源供應器？
    3. KEMET / Pulse 等併購資產是否真的帶來 AI server 交叉銷售？
    4. AI 相關產品毛利率是否高於集團平均？

    功率電感 / 磁性元件¶

    AI server VRM 多相電源架構提高功率電感需求。理論上功率電感與高功率 GPU 平台連動直接，但台廠是否受益取決於規格與 AVL。

    關鍵規格：

    • 高飽和電流 Isat。
    • 低 DCR。
    • 低損耗。
    • 溫升控制。
    • 一體成型或高可靠封裝。

    可驗證問題：

    1. 奇力新 / 美桓 / 國巨集團磁性元件是否進入 AI server power module 或 ODM AVL？
    2. 產品是一體成型功率電感、繞線型電感，還是一般磁珠 / EMI 元件？
    3. 是否已量產出貨，或仍在送樣？
    4. 競爭者如 TDK、Murata、Sunlord 是否已在同一客戶具備更高份額？
    5. 若市場傳出 Sunlord 進入 NVIDIA 供應鏈，需確認是直接供應、經 power module 供應，還是經 ODM 間接導入。

    電流感測電阻¶

    AI server power management 需要電流偵測、保護與監控。電流感測電阻可能受益，但台廠受益程度需驗證。

    可驗證問題：

    1. 大毅 / 國巨 / 華新科是否具備低阻值、低 TCR、高功率電流感測電阻規格？
    2. 是否通過 AI server power module 客戶認證？
    3. 毛利率是否高於標準厚膜電阻？
    4. 中國或日系競爭者是否在同規格更具成本或可靠度優勢？

    研究結論模板¶

    在建立 AI server thesis 前，至少要完成以下六格：

    公司：
品類：MLCC / 鉭容 / 功率電感 / 電流感測電阻 / 其他
規格：
供應鏈階段：未確認 / 送樣 / AVL / 小量量產 / 大量量產
營收占比：待確認 / 公司揭露 / 估算
毛利率：低於平均 / 接近平均 / 高於平均 / 待確認
風險：客戶集中 / 規格不足 / 中國競爭 / 日韓壓制 / 需求波動

    不建議直接引用的數字¶

    以下數字若無公司 IR、ODM、NVIDIA、生產商或高品質市調佐證，暫不寫入正式結論：

    • GB200 / NVL72 單機架 MLCC 總顆數。
    • AI server 高端 MLCC 精確市占率。
    • 單機架功率電感顆數。
    • AI server 被動元件 TAM / CAGR。
    • 供應鏈傳聞中的直接供貨關係。
