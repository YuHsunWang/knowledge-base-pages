    台灣被動元件公司比較¶

    關聯頁面：被動元件產業研究總覽、MLCC、晶片電阻與電感產品矩陣、被動元件景氣循環與追蹤框架。

    本頁是公司研究框架，不是投資建議。各公司最新產品別營收、應用占比、毛利率、稼動率、AI/車用占比需以 MOPS、公司 IR、法說逐字稿逐季更新。

    國巨（2327）¶

    • 定位：台灣被動元件龍頭；由 MLCC/電阻標準品廠轉型為全球多產品線被動元件平台。
    • 核心產品：MLCC、晶片電阻、鉭質電容、薄膜電容、鋁電解電容、電感/磁性元件、感測器、天線/通訊元件。
    • 策略：透過 KEMET、Pulse、Telemecanique Sensors 等併購擴張車用、工控、航太、醫療與高可靠度產品；整合奇力新補強電感與磁性元件。
    • 研究重點：
    • 高階/特殊應用占比是否提高。
    • 併購後整合費用、負債、商譽與現金流。
    • 車用、工控、AI server 營收是否可量化。
    • MLCC/電阻標準品價格循環對毛利率的影響是否降低。
    • 風險：併購整合、財務槓桿、標準品循環、商譽減損。
    • 來源入口：https://www.yageo.com/en/InvestorRelations

    華新科（2492）¶

    • 定位：台灣主要 MLCC 與晶片電阻廠；較純粹反映被動元件本業循環。
    • 核心產品：MLCC、晶片電阻、RF/保護元件、電感（待拆比重）。
    • 研究重點：
    • MLCC 稼動率、ASP、庫存天數與毛利率修復。
    • 車用/工控高階 MLCC 認證進展。
    • 中國消費電子與通訊需求回溫程度。
    • 產品 mix 是否脫離標準品價格競爭。
    • 風險：產品集中度高、標準品價格壓力、消費電子曝險。
    • 來源入口：https://www.passivecomponent.com/

    禾伸堂（3026）¶

    • 定位：被動元件製造 + 代理/通路混合型公司。
    • 核心產品/業務：MLCC 相關產品、自有品牌、代理銷售與其他電子零組件。
    • 研究重點：
    • 製造業務與代理業務營收/毛利拆分。
    • 代理通路庫存對營收與毛利率的影響。
    • 自有 MLCC 是否有利基規格與高階化。
    • 應收帳款與存貨週轉，避免通路庫存堆積。
    • 風險：代理毛利較低、代理線變動、通路庫存週期。
    • 來源入口：https://www.holystonecaps.com/

    大毅（2478）¶

    • 定位：晶片電阻專業廠。
    • 核心產品：厚膜晶片電阻、薄膜晶片電阻、電流感測電阻、高功率電阻、精密/抗硫化/車用電阻。
    • 研究重點：
    • 高階電阻占比，尤其電流感測、抗硫化、高功率、車規。
    • 車用、伺服器、工控應用出貨與客戶認證。
    • 標準厚膜電阻 ASP 與中國同業價格競爭。
    • 稼動率與毛利率彈性。
    • 風險：單一產品循環、標準品商品化、客戶集中度。
    • 來源入口：https://www.tai.com.tw/

    奇力新 / 美桓¶

    • 定位：台灣電感與磁性元件代表；奇力新已納入國巨集團脈絡，美桓為磁性元件供應商之一。
    • 核心產品：功率電感、晶片電感、磁珠、共模濾波器、變壓器、EMI 濾波元件。
    • 研究重點：
    • AI server VRM/電源模組功率電感需求是否量產導入。
    • 車用功率電感、BMS、ADAS、OBC 相關認證。
    • 國巨集團整合後交叉銷售與客戶滲透率。
    • 與 TDK、Murata、順絡電子等競爭關係。
    • 風險：資訊透明度、標準電感價格競爭、產品規格未達高階 AI server 要求。
    • 來源入口：https://www.maglayers.com/；奇力新官網待確認。

    公司比較結論¶

    • 國巨：平台型、產品線最完整，研究難度最高；重點在併購整合與高階產品 mix。
    • 華新科：MLCC/電阻循環 exposure 較高，適合當被動元件景氣溫度計。
    • 禾伸堂：製造與通路混合，需特別看代理庫存與毛利率。
    • 大毅：電阻專業廠，高階電阻升級是關鍵。
    • 奇力新/美桓：電感/磁性元件，AI server/車用電源是主要成長驗證點。

    每季更新欄位¶

    • 月營收 YoY/MoM。
    • 毛利率、營業利益率、EPS。
    • 存貨天數、存貨跌價。
    • 應收帳款週轉。
    • 稼動率與 ASP 法說表述。
    • AI server、車用、工控應用占比。
    • Capex 與產能策略。
    • 法說語氣：保守、中性、轉樂觀、過熱。

    深化版公司別研究追問¶

    第二輪深化研究後，公司比較頁新增「研究追問 + 紅旗風險」。以下內容是研究框架，不是已確認財務結論；最新營收、毛利率、估值、AI/車用占比仍需用 MOPS、TWSE、公司 IR、法說逐字稿逐季更新。

    國巨：平台型公司，重點在品類拆分與併購整合¶

    研究追問

    • AI 相關營收定義包含哪些品類？MLCC、鉭容、電感、電阻、感測器是否分開揭露？
    • KEMET / Pulse / 奇力新 / 感測器事業是否真的形成交叉銷售？
    • 高階品占比提升是否降低標準品景氣循環波動？
    • 淨負債、商譽、利息成本是否壓縮估值？
    • 車用與工控營收是否已量化？毛利率是否高於集團平均？

    紅旗風險

    • 平台敘事掩蓋標準品 ASP 壓力。
    • 併購整合費用、商譽減損或財務槓桿高於預期。
    • AI 相關占比未拆品類，導致市場誤讀。

    華新科：MLCC / 電阻循環純度較高¶

    研究追問

    • MLCC、晶片電阻、RF/保護元件各占比？
    • AI server / 車用是送樣、AVL 還是量產？
    • 標準品占比是否下降？
    • EPS 是否受到業外或一次性損益影響？
    • 新產能 ramp-up 是否帶來固定成本壓力？

    紅旗風險

    • MLCC 景氣回升被市場過度外推。
    • 業外收益使 EPS 品質被高估。
    • 中國消費電子與標準品價格競爭壓力未充分反映。

    禾伸堂：製造 + 代理雙軌，需拆營收品質¶

    研究追問

    • 製造與代理營收/毛利拆分？
    • 代理庫存是否領先反映景氣轉折？
    • 自有品牌 MLCC 是否有利基規格？
    • 毛利率變化來自 ASP、產品 mix、代理線、匯率，還是一次性因素？

    紅旗風險

    • 代理業務毛利率與製造業務混估。
    • 通路庫存堆積造成營收與應收帳款風險。
    • 估值若顯著高於同業，需確認毛利率改善是否具可持續性。

    大毅：電阻專業廠，高階化是核心 thesis¶

    研究追問

    • 高階電阻占比：電流感測、抗硫化、薄膜、高功率。
    • 是否進入車用、AI server power module 或工控客戶？
    • 標準厚膜電阻是否受中國廠報價壓力？
    • 稼動率與毛利率對 ASP 的彈性。

    紅旗風險

    • 標準厚膜電阻商品化。
    • 高階電阻占比不透明。
    • 車用 / AI 敘事停留在送樣，未轉成量產。

    奇力新 / 美桓：電感與磁性元件，重點在 AI VRM 與車用電源¶

    研究追問

    • 功率電感是否進入 AI server VRM 或車用 OBC / DC-DC？
    • 一體成型電感規格、良率與成本競爭力如何？
    • 國巨整合後是否提升客戶滲透率？
    • 與 TDK、Murata、Sunlord 在同客戶的定位差異？

    紅旗風險

    • 資訊透明度下降。
    • 順絡等中國競爭者往 AI server / 車用電感滲透。
    • 市場傳聞供應鏈導入未確認直接/間接、送樣/量產口徑。
